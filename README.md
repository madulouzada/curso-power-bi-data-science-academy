# Microsoft Power BI Para Data Science (DSA)

Este repositório contém o material prático, laboratórios e mini-projetos desenvolvidos durante o curso **Microsoft Power BI Para Data Science**, ministrado pela **Data Science Academy (DSA)**.

O objetivo deste repositório é documentar a minha jornada de aprendizagem na transição do Business Intelligence tradicional para a análise preditiva, utilizando técnicas de Data Science e Machine Learning integradas ao Power BI.

## 🎯 Objetivos do Curso
* **Fundamentos de BI:** Conexão, limpeza e transformação de dados (ETL) com Power Query.
* **Data Science:** Integração do Power BI com as linguagens **R** e **Python**.
* **Análise Preditiva:** Criação de modelos de Machine Learning para extração de insights.
* **Dashboards Avançados:** Desenvolvimento de visualizações interativas para diversas áreas de negócio.
